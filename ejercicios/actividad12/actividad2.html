background color - color de fondo

Si no se especifica un color de fondo, el fondo es trasparente.

CSS trata cada elemento HTLM como un cuadro y la propiedad background-color establece el color del fondo para el cuadro. Se puede especificar un color de fondo de las mismas tres formas en que puede especificar colores de primer plano: Valores RGB, codigo Hexadecimal y Nombre de colores. Tambien hemos utilizados la propiedad padding para separar el texto de los bordes de los cuadros. Esto hace que sea mas facil de leer.
