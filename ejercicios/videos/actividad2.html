Un video de un cachorro jugando en la nieve

la etiqueta video: para especificar la ubicacion del archivo que se reproducira, puede usar el elemento <source> dentro del elemento <video> (esto debe reemplazar el atributo src en la etiqueta <video> de apertura.) Tambien puede utilizar varios elementos <source> para especificar que el video esta disponible en diferentes formatos. (debido a un error en el ipad, debe proporcionar el video MP4 como el primer formato. De lo contrario, puede que no inicie.)

src: este atributo especifica la ruta de acceso al video,

type:se debe utilizar este atributo para decirle al navegador que formato es el video. De lo contrario, descargara parte del video para ver si puede reproducir el archivo (que tomara tiempo y ancho de banda).

codecs:El codec que se usó para codificar el vídeo se suministrara dentro del atributo type. tenga en cuenta el uso de las comillas simples, asi como de las comillas dobles en el atributo de tipo, cuando se suministra
