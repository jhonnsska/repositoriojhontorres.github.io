Adding Html5 video

Un video de un cachorro juegando en la nieve

En HTLM5 no es necesario proporcionar valores para todos los atributos, como los controles, reproduccion automatica y atributos de bucle utilizados con el elemento <video>. estos atributos son como interruptores de encendido/apagado. si el atributo esta presente, activa esa opcion. si se omite el aatributo, la opcion esta desactivada. Si el navegador no admite el elemento <video> de apertura y las etiquetas de </video> de cierre.

preload: este atributo le indica al navegador que hacer cuando se carga la pagina. Puede tener uno de los tres valores:
none:el navegador no debe cargar el video hasta que el usuario presiona play.
metadata: el navegador debe simplemente recopilar informaccion como el tamaño, el primer fotograma, lalista de psitas y la duración,
