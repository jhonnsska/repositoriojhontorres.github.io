Adding HTLM5 audio

<audio> HTLM5 introdujo el elemento <audio> para incluir archivos de audio en sus páginas. Al igual que con el video HTLM5, los navegadores esperan diferentes formatos para el audio. El elemento <audio> tiene un numero de atributos que le permiten controlar la reproduccion de audio:

src:este atributo especifica la ruta de acceso al archivo de audio.

controls: este atributo indica que si el reproductor debe mostral controles. si no utiliza este atributo, no se mostrarán controles de forma predeterminada. Tambien puede especificar sus propios controles mediante JavaScript.

autoplay:la presencia de este atributo inidca que el audio debe empezar a reproducirse automaticamente. (Se considera una mejor práctica permitir que los visitantes elijan reproducir el audio.)

loop: este atributo especifica que la pista de audio debe reproducirse de nuevo una vez haya finalizado
