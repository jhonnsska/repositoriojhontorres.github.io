Este ejemplo combina muchas de las tecnicas mostradas en los ejercicios

Hay propiedades para controlar el tipo de fuente, el tamaño, la ubicacion, el estilo y el espaciado.

Caracteristicas

puede controlar el espacio entre lineas de texto, las letras individuales y las palabras. El texto también se puede alinear a la izquierda, derecha, centrada o justificada. Tambíen puede tener sangria.
